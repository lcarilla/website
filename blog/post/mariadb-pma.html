Install MariaDB and PhpMyAdmin on Debian 11
This guide assumes that you have installed Debian 11 with the Apache Web server
After you have connected to your server, install MariaDB and enable it 

apt install mariadb-server
systemctl enable --now mariadb
Setting Up MariaDBNow enter this command in your terminal to configure your MariaDB instance and select your settings 

mysql_secure_installation
Set up a database userRun the mariadb command to be able to communcate with the DBMS 

mariadb
Then create a new user and grant all privileges to them 

CREATE USER 'admin'@'%';
GRANT ALL ON *.* TO 'admin'@'%' IDENTIFIED BY 'password' WITH GRANT OPTION;
FLUSH PRIVILEGES;
exit;
PMA installInstall the dependecies 

apt install wget php php-cgi php-mysqli php-pear php-mbstring libapache2-mod-php php-common php-phpseclib php-mysql
Then install PMA itself 

apt install phpmyadmin
Select the Apache Webserver if a prompt from the installer pops upYou should have PhpMyAdmin running on your webserver (example.com/phpmyadmin) now
